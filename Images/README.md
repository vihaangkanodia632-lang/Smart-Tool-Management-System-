## How to Take Screenshots (Windows)

- **Full screen:** `Win + PrintScreen` → saved to `Pictures/Screenshots/`
- **Selected area:** `Win + Shift + S` → crop → paste in Paint → save as PNG
- **Browser screenshot:** F12 → Ctrl+Shift+P → type "screenshot" → Full page screenshot

## How to Export the Flowchart as PNG

1. Open `Documentation/system_flowchart.html` in Chrome
2. Press `Ctrl+Shift+I` (DevTools) → `Ctrl+Shift+P`
3. Type `Capture full size screenshot` → Enter
4. Save the downloaded PNG as `system_flowchart.png` in this folder

## Image Guidelines for GitHub

- Use **PNG** for screenshots (lossless, sharp text)
- Use **JPG** for hardware photos (smaller file size)
- Keep images under **2MB** each for fast loading
- Minimum resolution: **1280×720** for screenshots
