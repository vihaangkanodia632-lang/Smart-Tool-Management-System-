# Documentation — Smart Tool Management System

This folder contains the full project documentation.

## Files

| File | Description |
|---|---|
| `Smart_Tool_Management_Report.docx` | Complete project report (Word document) — includes hardware details, wiring tables, Firebase setup, all dashboard screenshots, test results, and conclusion |
| `system_flowchart.html` | Full interactive system flowchart — open in any browser to view the complete hardware + software + cloud flow |

## Report Contents

The project report covers:

1. **Project Overview** — objectives and system architecture
2. **Hardware Components & Wiring** — component table, pin connections, hardware photo
3. **Software Architecture** — technology stack, MicroPython code structure
4. **Firebase Database Structure** — all 5 database nodes explained
5. **Web Dashboard Screenshots** — all 10 live screenshots with captions
6. **Registered Workers & Tools** — tables with real RFID UIDs
7. **Test Results** — successful operations, edge cases, issues found and resolved
8. **Conclusion** — key achievements and future enhancements

## Viewing the Flowchart

1. Download `system_flowchart.html`
2. Open in **Google Chrome** or **Microsoft Edge**
3. To export as image: `Ctrl+Shift+I` → `Ctrl+Shift+P` → type `Capture full size screenshot`

## Viewing the Report

Open `Smart_Tool_Management_Report.docx` with:
- **Microsoft Word** (best formatting)
- **Google Docs** (upload to Drive → Open)
- **LibreOffice Writer** (free alternative)
